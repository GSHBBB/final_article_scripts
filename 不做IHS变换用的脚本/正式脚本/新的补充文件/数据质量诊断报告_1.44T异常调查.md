# 数据质量诊断报告：爱尔兰1.44万亿美元数字贸易出口异常

## 执行摘要

**问题**：爱尔兰2024年数字服务（ICT/SI）出口额被报告为1,449,400百万美元（1.44万亿），远高于合理预期。

**根本原因**：OECD BaTIS数据存在**重复记录**，其中部分国家-对方地区组合有3条完全相同或接近的观测值被全部累加。

**修复方案**：对于同一(出口国, 对方地区, 行业, 年份)组合，只保留一条记录。

---

## 详细诊断

### 1. 原始数据构成

**爱尔兰2024年SI记录数据分解：**

| 数据来源 | 记录数 | 总出口值（百万美元） | 占比 |
|---------|-------|------------------|------|
| METHODOLOGY_TYPE=NaN | 249 | 1,309,280 | 90.3% |
| METHODOLOGY_TYPE=ME | 146 | 63,079 | 4.4% |
| METHODOLOGY_TYPE=IE | 30 | 75,417 | 5.2% |
| METHODOLOGY_TYPE=AG | 1 | 1,624 | 0.1% |
| **合计** | **426** | **1,449,400** | **100%** |

### 2. 重复记录识别

**关键发现：**
- 207个对方地区中，177个各有1条记录
- 但有24个对方地区各有**3条相同或接近的记录**

**具体示例：**

#### 示例1：World
```
Counterpart area: World
记录1：OBS_VALUE = 315,264.19
记录2：OBS_VALUE = 315,264.19 （完全相同）
记录3：OBS_VALUE = 175,065.00 （接近）
小计：805,593 百万美元（实际应为 315,264 百万美元）
过度计算倍数：2.55倍
```

#### 示例2：Finland
```
Counterpart area: Finland
记录1：1,534.67
记录2：1,500.97
记录3：1,237.27
小计：4,272.91 百万美元（实际应为～1,500 百万美元）
过度计算倍数：~2.8倍
```

#### 示例3：Germany  
```
Counterpart area: Germany
记录1, 2, 3：三条不同值的记录
小计：64,186.32 百万美元
过度计算倍数：~3倍
```

### 3. 重复数据影响量化

**按对方地区记录数分类：**
- 1个记录的对方地区：177个，总和 72,880 百万美元
- 3个记录的对方地区：24个，总和 1,236,400 百万美元 → 应为 412,133 百万美元

**重复部分：** 1,236,400 - 412,133 = **824,267百万美元**（占原始值56.8%）

### 4. 修正后的数据

**去重方案**：对每个(出口国, 对方地区, 行业, 年份)组合，只保留第一条记录

**修正后爱尔兰SI数据：** 约 485,013 百万美元（= 72,880 + 412,133）

**修正倍数：** 485,013 / 1,449,400 = 33.4%（即原始数据被过度计算了3倍）
